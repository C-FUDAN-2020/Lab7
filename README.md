### 问题一
小明同学通过如下所示的代码来对若干数据进行求和。但是小明测试时发现如果输入的5个数据中最后一个数据是5，计算结果不会出现错误；否则计算结果就是错误的。测试结果如图所示。请帮他分析产生这样的现象的原因。

```c
#include <stdio.h>

int main(){
	int num[5];
	int sum = 0;
	for(int i = 1; i<= 5; i++){
		scanf("%d",&num[i]);
		printf("Number %d: %d\n",i,num[i]);
	}
	for(int i = 1; i <= 5; i++){
		sum += num[i];
	}
	printf("The sum is %d.",sum);
	return 0;
}
```
